A **complete Python implementation** of the **Lox** programming language from the book  
[*Crafting Interpreters*](https://craftinginterpreters.com) by Robert Nystrom.

---

## Features

- **Full Lox language support**: variables, functions, classes, control flow
- **Semicolon-free syntax**: newlines act as statement terminators  
  ```lox
  print(1 + 2)
  var x = 5
  print(x * 2)
